# 🍳 Cooking To-Do Planner

> A smart meal planning web application that generates personalized meal plans, grocery lists, budget analysis, ingredient substitutions, and cooking timelines.

---

## 🌟 Overview

Cooking To-Do Planner helps users organize their daily cooking activities by generating meal recommendations based on:

- 👨‍👩‍👧‍👦 Number of People
- 💰 Budget
- ⏱️ Available Cooking Time
- 🥗 Dietary Preferences
- 🛒 Available Ingredients

The application then creates a complete cooking workflow including meal suggestions, grocery requirements, substitutions, and task planning.

---

## ✨ Features

### 🍽️ Meal Planning
- Personalized Breakfast, Lunch, and Dinner recommendations
- Supports multiple dietary preferences
- Time-based meal selection

### 🛒 Smart Grocery List
- Detects missing ingredients
- Categorized shopping lists
- Reduces unnecessary purchases

### 🔄 Ingredient Substitutions
- Alternative ingredients for unavailable items
- Flexible meal preparation

### 💰 Budget Analysis
- Estimated meal cost calculation
- Budget feasibility checking
- Cost-aware recommendations

### 📋 Cooking Timeline
- Step-by-step cooking workflow
- Organized task scheduling
- Improved kitchen efficiency
